#  Construction Project Performance & Risk Analytics Dashboard

This project analyzes **1000 construction projects** to evaluate **project delays, budget performance, contractor efficiency, and milestone bottlenecks** using **Power BI**.

The goal of this dashboard is to help project managers and decision-makers identify **high-risk projects, operational inefficiencies, and schedule management issues**.

---

#  Project Overview

Construction companies often manage multiple large projects simultaneously. Delays and cost overruns can significantly impact profitability and delivery timelines.

This dashboard provides a **data-driven approach to monitoring project performance**, enabling stakeholders to quickly detect risks and improve project planning.

---

#  Tools Used

- Microsoft Power BI
- Power Query (Data Cleaning)
- DAX Measures
- Excel Dataset

---

#  Dataset

The dataset contains **1000 construction project records** with the following information:

- Project Name
- Contractor
- Planned Progress %
- Actual Progress %
- Delay Days
- Project Budget
- Cost to Date
- Milestone Stage
- Completion Status
- Budget Status

---

#  Key Findings

• **93.7% of projects are within budget**, indicating strong overall cost control across the project portfolio.

• However, projects experience an **average delay of ~38 days**, highlighting schedule management challenges.

• A small number of **large infrastructure projects contribute the majority of delays**, including projects such as metro stations and infrastructure zones.

• **Contractor performance varies significantly**, with certain contractors showing consistently higher delay counts.

• **Foundation and Structure phases show higher delay patterns**, indicating operational bottlenecks during early construction stages.

• In many cases, **Actual Progress lags behind Planned Progress**, suggesting gaps between project planning and execution.

• **Large-scale infrastructure projects show higher schedule risk**, likely due to complexity and coordination requirements.

• Some projects finish **ahead of schedule**, suggesting that efficient project management practices exist and could be replicated across other projects.

---

#  Business Recommendations

Based on the analysis, the following actions could improve project performance:

- Implement **stronger milestone monitoring** to reduce schedule delays.
- Introduce **contractor performance evaluation systems**.
- Allocate additional resources during **high-risk construction phases** such as foundation and structural work.
- Use **early delay indicators** (progress gap analysis) to detect project risks sooner.
